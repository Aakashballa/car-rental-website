# 🚗 Car Rental Website

A responsive frontend web application for a car rental service, built using HTML, CSS, and JavaScript.

---

## 🚀 Features

- 📍 Search form for rental location and dates
- 🚘 Display of multiple vehicle categories
- 📱 Fully responsive design for all devices
- 🎨 Clean and modern UI
- ⚡ Interactive navigation and UI components

---

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript
